# @agentmirror/web

The AgentMirror marketing landing page — Next.js (App Router) + Tailwind v4, dark/neon.

```bash
npm run dev   -w @agentmirror/web    # http://localhost:3000
npm run build -w @agentmirror/web    # production build
```

## Deploy (Vercel)

Point Vercel at this repo with **Root Directory = `apps/web`** (it auto-detects Next.js). No env vars needed for the landing page.

This is the seed of the full web app from the tech stack; the dashboard + billing (Phase 3) land here later. The interactive [playground](../playground) can be embedded once both are deployed.
